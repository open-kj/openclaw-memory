# HEARTBEAT.md - 主动模式

## 主动检查清单（每次心跳执行）

### 1. 持仓状态检查（每30分钟一次）
- 读取 `SESSION-STATE.md` 获取最新持仓
- 计算炬芯科技(sz300223)距止损线距离
- 如果距止损 <3% → 立即推送飞书预警
- 如果距止损 <5% → 下次心跳时提醒

### 2. 大盘异动检查（每2小时一次）
- 如果大盘涨跌 >1.5% → 立即推送飞书并分析原因

### 3. 持仓股票新闻检查（每小时一次）
- 用 Tavily 搜索持仓股票最新消息
- 如果有重大利空 → 立即推送飞书预警

### 4. 现金水位检查（每次心跳）
- 如果现金 <50,000元 → 提醒不开新仓
- 如果现金 >300,000元 且 有强势回调机会 → 主动提出分析

### 5. 自我学习记录（每3小时一次）
- 记录最近判断对/错的要点到 `memory/YYYY-MM-DD.md`

## 主动触发规则

| 条件 | 动作 | 方式 |
|------|------|------|
| 炬芯科技距止损<3% | 立即预警 | 飞书推送 |
| 大盘涨跌>1.5% | 主动分析 | 飞书推送 |
| 持仓股重大新闻 | 立即推送 | 飞书推送 |
| 止盈达标+10% | 提醒用户决策 | 飞书推送 |
| 连续3天无操作 | 主动做复盘 | 飞书推送 |
