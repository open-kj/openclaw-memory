### 教训2：持仓数据必须同步
- 日期: 2026-03-20
- 错误: cron任务用旧持仓（5只），实际只有2只
- 正确: 交易后立即更新所有位置（MEMORY.md + cron任务 + handbook）

### 教训3：cron delivery格式导致发送失败
- 日期: 2026-03-21
- 错误: `deliver: true` 写进 message 字段，导致 target 变成 "~"
- 正确: 使用 `announce` 模式 + `delivery.channel` + `delivery.to`
- 状态: ✅ 已修复

### 教训4：每次计算必须包含验证数据
- 日期: 2026-03-20
- 错误: 瞎编收益率（+5.26%，实际+1.74%）
- 正确: 精确计算 → 写清楚过程 → 欢迎用户验证
- 状态: ✅ 已修复
