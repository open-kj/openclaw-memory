## 记忆系统维护规则（重要）

### 每次操作后必须同步更新
1. **交易后**：立即更新 MEMORY.md + cron任务 + handbook
2. **学习后**：把重要内容合并到 MEMORY.md
3. **发现错误**：立即纠正并记录到 MEMORY.md 的教训区
4. **每次改动后**：git add + commit + push 到 open-kj/openclaw-memory

### cron job 规则
- 周一到周五用 `1-5`，不能裸写 `* * *`
- 周中和周末同一时间点的job不能同时存在
- job里写清楚持仓数据（不要用"持仓数据同上次"这类模糊描述）
