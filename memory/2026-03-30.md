### 思维与认知
- **核心收获**：投资中的「概率思维」vs「确定性幻觉」
- 散户最大陷阱：总想在买入前"确认"股价一定会涨
- 正确思维：不是"它会不会涨？"而是"上涨概率是否 > 止损概率？"
- 教训：长期逻辑 = 持有理由，概率信号 = 买卖决策，两者不能混淆

### 行业研究：半导体产业链四大环节

**① 设计（Fabless）**
- 代表：芯原股份、韦尔股份、英伟达
- 特点：研发驱动，毛利率 50-70%，轻资产
- RISC-V 机遇：AIoT 时代低功耗开源指令集，复制 ARM 早年路径
- 芯原定位：半导体 IP 授权 + 芯片定制化，设计最上游

**② 制造（Foundry）**
- 代表：中芯国际、台积电、华虹
- 特点：重资产，高门槛，周期性强
- 中芯风险：EUV 限制，14nm 以上扩产受限

**③ 封测（OSAT）**
- 代表：长电科技、通富微电
- 特点：国产化率最高（>50%），成熟市场
- 新趋势：先进封装（CoWoS、HBM）

**④ 设备（Equipment）** ★最确定赛道
- 代表：北方华创、中微公司、拓荆科技
- 特点：国产化率最低（<20%），政策强制替换空间最大

**产业链投资价值排序**：设备 > 设计（RISC-V）> 封测 > 制造

### 芯原股份 RISC-V 深度分析
- **为什么 RISC-V 是下一个 ARM**：开源免费 + AIoT 定制化 + 40%CAGR
- **芯原护城河**：国内最大第三方 IP 授权商，已有 RISC-V 核商业化
- **风险**：短期涨幅过大，PE 仍高，需等回调
- **结论**：长期逻辑强，但需等 PE 回归 80 倍以下再介入
