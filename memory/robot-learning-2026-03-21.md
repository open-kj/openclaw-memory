# 机器人互相学习 - 2026-03-21

*学习来源：分析71个已安装技能 + 对比不同AI系统的交易决策方式*

---

## 一、从量化交易系统学到的

### 1.1 机器学习的择时 vs 人工判断

**其他AI系统做法：**
- 使用历史数据训练模型，预测未来涨跌概率
- 严格的风控模块：最大回撤、止损线、仓位管理
- 情绪 neutrality：无情绪波动，100%机械执行

**我发现的问题：**
- 我的止损规则（-5%）已经类似，但执行依赖 cron 任务，有15分钟延迟
- 人工在恐慌时可能不止损，但我的 cron 会强制执行
- **改进方向**：可以在收盘汇报中增加「触发预警」功能

### 1.2 多因子选股模型

**其他AI系统做法：**
- 价值因子（PE、PB、PS）
- 成长因子（净利润增速、营收增速）
- 质量因子（ROE、现金流、负债率）
- 动量因子（近N日涨幅）

**我已经做的（手册v2.0）：**
✅ 四步筛选流程已包含这三类因子
❌ 但执行是「预判」而非「实时计算」

### 1.3 仓位管理

**其他AI系统做法：**
- 凯利公式：f = (bp - q) / b
- 固定比例：单只不超过总仓位的20%
- 金字塔式加仓：越跌越买（或越涨越买）

**我的现状：**
- 当前持仓4只，最重的天孚/炬芯约18万，占总资金18%
- 没有明确的「单只仓位上限」规则

**学习结论**：加入规则：「单只股票最大仓位 ≤ 20%」

## 二、从其他 Skills 学到的

### 2.1 Polymarket 的风险管理思维

Polymarket skill 的设计非常优雅：
- 每个预测市场有明确的「概率」和「时间」
- 设置 alert，当概率变化超过阈值时触发

**可借鉴**：
在股票监控中加入「概率预警」——比如「炬芯科技跌破止损概率超过60%」时提前告警

### 2.2 tech-news-digest 的信息聚合

这个技能能将6个来源的信息汇总去重排序。

**可借鉴**：
在每天的盘前分析中，也应该有一个「多源信息聚合」流程：
- 腾讯财经（价格）
- 东方财富（资讯）
- 雪球/同花顺（舆情）
- 央视/政策（宏观）

### 2.3 tradingagents-analysis 的分工模式

这个技能用「15个AI分析师」分别做不同维度的分析。

**可借鉴**：
我目前的分析是「单兵作战」，但我可以模拟这种分工：
- 一个子任务分析技术面
- 一个子任务分析资金面
- 一个子任务分析情绪面
- 最后汇总

## 三、目前AI做交易的主要流派

### 3.1 纯量化（完全不用AI）
- 规则驱动：IF 价格<止损线 THEN 卖出
- 无预测，纯粹执行
- **代表**：散户的「条件单」

### 3.2 AI辅助决策（我目前的方式）
- AI 提供分析建议，人类做最终决策
- 保留人的判断力
- **优点**：结合AI速度 + 人类直觉
- **缺点**：人可能不执行AI建议（损失厌恶）

### 3.3 全自动AI交易
- AI 100%自主决策并执行
- 无情绪干扰
- **问题**：黑天鹅事件无法处理

**我的定位**：第2种（AI辅助决策）

## 四、关键差距与改进

### 差距1：没有回测能力
- 其他AI系统可以回测「过去3年如果用这个策略能赚多少」
- 我目前是「边做边学」，无法评估策略有效性
- **改进**：建议用户做一次历史回测（用真实数据模拟）

### 差距2：没有风控仪表盘
- 其他系统有实时「当前风险值」「最大回撤」
- 我只有「止损线」这一个指标
- **改进**：增加「组合风险值」概念

### 差距3：信息滞后
- 其他AI系统可以实时监控新闻、公告、社交媒体
- 我的 cron 是每15分钟一次，无法捕捉突发新闻
- **改进**：盘前和盘后收集信息，不依赖盘中实时

## 五、核心收获

### 从「机器人互相学习」中得到的5条行动清单

1. **增加「单只仓位上限20%」规则** — 防止重仓单只
2. **加入「概率预警」** — 不是等跌破止损线，而是当概率超过阈值时提前告警
3. **每日盘前「多源信息聚合」** — 用3个以上数据源验证判断
4. **模拟「多分析师分工」** — 每次决策从3个维度分析
5. **建议用户做一次历史回测** — 评估当前策略在过去的市场中的表现

## 六、我能给其他AI系统的建议

如果其他AI系统要帮助散户做股票交易：

1. **不要给太多选择** — 选择越多越容易什么都不做
2. **规则要简单** — 「跌破5%就卖」比「综合考虑基本面和技术面后决定」更有效
3. **必须包含「退出机制」** — 告诉用户「什么时候必须卖」比告诉「什么时候可以买」更重要
4. **承认自己的局限性** — AI无法预测黑天鹅，不要过度自信
