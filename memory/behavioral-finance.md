## 一、8大认知偏差与克服方法

### 1. 损失厌恶（Loss Aversion）
**定义：** 亏损的痛苦是同等盈利快乐的2.5倍

**实战表现：**
```
场景：持仓亏损-3%，我的反应
❌ 不愿意止损，想等反弹回来
✅ 记住：亏损100元需要盈利111元才能回本
✅ 亏损-5%立即止损，不犹豫
```

**克服方法：**
- 止损线已固化（-5%），机械执行
- 忘掉买入价，忘掉成本
- 问自己：如果现在空仓，我会买这只吗？

---

### 2. 锚定效应（Anchoring）
**定义：** 过度依赖第一个信息（买入价）

**实战表现：**
```
场景：北京君正现价119.80，成本122.34
❌ "我已经亏了2.54元，等涨回本再卖"
✅ "如果现在119.80，我会买吗？"（不会）
✅ 忘掉成本，按现价和止损线决策
```

**克服方法：**
- 成本价是沉没成本，不应影响决策
- 决策只看：现价 + 止损线 + 基本面

---

### 3. 羊群效应（Herding）
**定义：** 盲目跟随大多数人的行为

**实战表现：**
```
场景：所有人都在讨论AI股票
❌ "大家都在买，我也买"
✅ 检查自己的止损线和仓位
✅ 如果追高买北京君正，现在已亏-2%
```

**克服方法：**
- 大多数人讨论时，可能是卖点
- 独立思考：这只股票值得持有吗？

---

### 4. 过度自信（Overconfidence）
**定义：** 高估自己的判断能力

**实战表现：**
```
场景：觉得自己能预测明天大盘
❌ "我觉得会反弹，加仓！"
✅ 数据说话："大盘已跌1.24%，跌破4000，空头信号"
✅ 少操作，多研究
```

**克服方法：**
- 每日复盘：我的判断对了几次？
- 连续3次判断正确后再提高信心

---

### 5. 确认偏误（Confirmation Bias）
**定义：** 只看支持自己观点的信息

**实战表现：**
```
场景：持仓北京君正
❌ 只看"AI概念"利好，忽略"PE 182倍"利空
✅ 同时看利好和利空
✅ 北京君正：利好AI + 利空高估值 + 非核心
```

**克服方法：**
- 主动寻找反对意见
- 问自己："什么情况下我会止损？"

---

### 6. 近期偏误（Recency Bias）
**定义：** 过度重视近期事件

**实战表现：**
```
场景：北京君正03-19刚涨了
❌ "前几天还在涨，没事"
✅ 查看5日线：已从129.66跌到119.80（-7.6%）
✅ 连续下跌5天，这是趋势
```

**克服方法：**
- 看长期数据，不只看今天
- 北京君正：连续5天下跌，不是"暂时调整"

---

### 7. 处置效应（Disposition Effect）
**定义：** 卖盈持亏

**实战表现：**
```
场景：
❌ 天孚通信赚了+5%，赶紧卖
❌ 北京君正亏了-2%，死拿等回本
✅ 盈利多的反而要持有（天孚强逻辑）
✅ 亏损多的反而要止损（北京君正风险高）
```

**克服方法：**
- 强的持仓多持，弱的持仓快跑
- 卖掉"对的"，持有"错的"是散户最大错误

---

### 8. 模糊厌恶（Ambiguity Aversion）
**定义：** 厌恶不确定性，宁可要已知的坏结果

**实战表现：**
```
场景：北京君正距止损3%
❌ "也许会反弹，不动"
✅ 模糊厌恶：-3%不确定，止损-5%是确定的
✅ 与其等不确定的反弹，不如执行确定的止损
```

**克服方法：**
- 止损是已知的坏事，不止损是未知的坏事
- 执行已知的规则，不赌未知的可能

## 二、实战决策检查清单

每次操作前问自己：

```
□ 1. 我有实时数据吗？（不看记忆中的价格）
□ 2. 这个决策符合止损规则吗？（-5%无条件）
□ 3. 我有没有验证过数字？（成本/现价/止损）
□ 4. 这是"想"操作还是"该"操作？
□ 5. 如果错了，最坏结果是什么？（能承受吗？）
□ 6. 多数人会怎么想？（逆向思考）
□ 7. 我是不是在等回本？（锚定效应）
□ 8. 我是不是因为害怕而不动？（损失厌恶）
```

## 三、散户最常见的错误

| 错误 | 克服方法 |
|------|---------|
| 频繁交易 | 设好止损，不动如山 |
| 卖盈持亏 | 盈利的多持有，亏损快跑 |
| 追涨杀跌 | 只买回调，不追高 |

| 过度自信 | 数据说话，不凭感觉 |
