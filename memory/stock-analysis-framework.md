## 选股分析框架

### 选股标准（逐步完善）
1. **基本面**：市值50-500亿、市盈率合理、业绩增长
2. **技术面**：上升趋势、成交量活跃、回调不破支撑
3. **资金面**：主力资金持续流入、筹码集中
4. **行业**：政策支持、景气度高

### 分析流程
1. 获取今日涨幅榜 → 筛选强势股
2. 获取资金流向 → 筛选主力买入
3. 获取K线 → 判断趋势
4. 获取新闻 → 验证逻辑
