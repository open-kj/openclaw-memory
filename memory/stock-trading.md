# A股模拟交易系统 - 执行规则

## 止盈止损规则
| 规则 | 条件 | 操作 |
|------|------|------|
| **止损** | ≤ -5% | 由我决定抛售 |
| **止盈提醒** | ≥ +10% | 提醒你，由你决定 |

## 监控频率
- **开盘后**：每15分钟监控一次
- **交易时间**：09:30 - 15:00

## 汇报内容
1. **现金**：当前可用资金
2. **持仓**：持仓股票涨跌情况
3. **操作建议**：是否触发止盈/止损

## 汇报模板
```
【持仓监控】
现金：XXX万
持仓：
- 股票A：+X.XX%  (持有X天)
- 股票B：-X.XX%  (持有X天)
...
总盈亏：+X.XX%

【操作提醒】
- 股票A 达到止盈线 → 等待用户决定
- 股票B 触及止损线 → 自动处理
```
