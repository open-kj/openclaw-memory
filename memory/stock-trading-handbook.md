# 股票交易执行手册 v2.0

## 核心原则
1. **数据源**: 腾讯财经API `qt.gtimg.cn` 实时数据
2. **计算基准**: 持仓成本（买入价 × 数量）+ 现金
3. **公式**: 总资产 = 持仓市值(实时) + 现金 | 总盈亏 = 总资产 - 1,000,000
4. **必须验证**: 每次计算后自验，确保100%准确
5. **止损规则**: -5% 无条件自动执行，不等待确认，不犹豫
6. **止盈规则**: +10% 提醒用户决定

## 三、计算流程（每15分钟执行）

### Step 1: 获取实时数据
```
API: https://qt.gtimg.cn/q=sz300394,sz300223,sz300548,sh688521
方法: Invoke-RestMethod
解析: 第4个~分割的字段为现价
```

### Step 2: 逐行计算持仓市值
```
天孚通信 = 现价 × 618
炬芯科技 = 现价 × 1530
博创科技 = 现价 × 1000
源杰股份 = 现价 × 1000
```

### Step 3: 计算每只股票盈亏
```
盈亏 = 市值 - 成本金额
收益率 = (现价 - 成本价) / 成本价 × 100%
```

### Step 4: 计算总资产
```
持仓总市值 = 4只股票市值之和
总资产 = 持仓总市值 + 294,992(现金)
总盈亏 = 总资产 - 1,000,000
```

## 四、股票筛选流程（买入前必做）

### 1. 初步筛选 ✅
通过**价值因子、成长因子、质量因子**筛选，挑选市场中兼具安全边际与增长潜力的优质股票。

### 2. 基本面深度分析 ✅
对候选股票进行财务健康验证：
- 盈利趋势：近3年净利润变化
- 现金流走向：经营现金流是否健康
- 资产负债结构：负债率、资产质量
- 核心盈利指标：ROE、毛利率、净利率变化

### 3. 资金动向分析 ✅
结合大盘走势、板块轮动、市场情绪，深度分析：
- **北向资金**：净流入/流出规模、持续时间
- **机构资金**：配置方向、重点个股
- **游资**：热点题材、短线炒作痕迹
- **量价指标**：量比、换手率、委比、内外盘
- **筹码分布**：是否集中、交易活跃度
- **盘口分析**：分时图、盘口挂单变化
- **资金意图**：是否存在对倒、诱多、诱空迹象

### 4. 风险评估 ✅
综合评估以下风险并设定触发条件：
- 政策风险：行业政策变化对公司的影响
- 行业周期风险：行业处于周期哪个阶段
- 公司经营风险：管理层、主营业务、竞争格局
- 流动性风险：大股东减持、解禁、融券做空

## 五、历史错误记录

### 错误1: 现金计算错误
- 日期: 2026-03-19
- 错误: 使用固定50,000元作为现金
- 正确: 现金=54,576元（需根据买卖记录计算）
- 教训: 现金必须从交易记录实时推算

### 错误2: 总资产计算错误
- 错误: 用持仓成本+现金
- 正确: 用持仓市值(实时)+现金
- 教训: 持仓市值必须用实时行情，不能用成本

### 错误3: 持仓数据未同步
- 日期: 2026-03-20
- 错误: cron任务使用旧持仓数据（5只），实际只有2只
- 教训: 交易后必须立即更新cron任务和记忆文件
- 教训: 每次交易后同步更新 MEMORY.md、cron任务、handbook
