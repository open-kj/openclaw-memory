## 止损规则（自动执行→推荐→用户决策）

**Step 1：止损触发**
- 现价≤止损价 → 自动卖出，不等待确认
- 飞书立即通知用户止损执行结果
- 止损后现金保留，不自动买入

**Step 2：止损后操作（新增）**
- 止损卖出后，我立即扫描候选池
- 5分钟内给出1-3只候选推荐股（含理由、目标价、止损线）
- 用户决定买哪只，我执行

**Step 3：执行新仓**
- 用户确认后立即执行
- 新仓同步更新：MEMORY.md + HEARTBEAT.md + cron任务 + Git

**候选池来源：**
- 芯片/光模块/AI算力主线（熟悉的赛道）
- 只选有认知、高流动性、低估值的标的
- 不追高、不买小市值、不买亏损股
